# AIDE (AI Development Ensemble) - Capstone Project

A multi-agent AI system for collaborative web application development built for Google's 5-Day AI Agents Intensive Course.

## Features
- 10 specialized AI agents working in coordination
- Evolutionary requirements gathering
- Enterprise-grade architecture
- Google ADK integration

## Setup
1. Clone repository
2. Create `.env` file with your `GOOGLE_API_KEY`
3. Run `pip install -r requirements.txt`
4. Execute `python main.py`

## Architecture
- **Requirements Evolver**: Evolutionary Q&A process
- **Frontend Ensemble**: UX, UI, and Frontend specialists
- **Backend Ensemble**: Data, API, and DevOps specialists
- **Quality Tier**: Validation, Code Quality, and Integration agents
